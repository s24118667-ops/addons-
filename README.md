# Create Aeronautics (Bedrock/MCPE scaffold)

Expanded scaffold for Bedrock Edition using `create_aeronautics` namespace.

## Included
- 19 items
- 22 blocks (including optional brass/copper/iron hull, wing, canopy variants)
- 5 starter recipes
- Texture key entries for every item and block
- Localization entries for every item and block

## Intentionally not included yet
- PNG textures
- Custom geometry/models
- Advanced moving-airship logic

## Packs
- `create_aeronautics_bp/` behavior pack
- `create_aeronautics_rp/` resource pack

Upload textures later using the exact filenames referenced in:
- `create_aeronautics_rp/textures/item_texture.json`
- `create_aeronautics_rp/textures/terrain_texture.json`
